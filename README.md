# Microsoft 365

Static HTML5 end-user guide for Microsoft Teams, OneDrive, and SharePoint. The guide is designed for GitHub Pages and can be hosted from the repository root without a build step.

## Files

- `index.html` - the full guide content.
- `styles.css` - layout, responsive design, print styles, and theme styles.
- `app.js` - navigation sorting, section linking, search, workflow hotspots, tabs, lightbox zoom, clickable tiles, theme toggle, and print controls.
- `assets/` - synthetic product-style screenshots and workflow graphics.
- `.nojekyll` - prevents GitHub Pages from running Jekyll processing.
- `404.html` - redirects unmatched GitHub Pages routes back to the guide.

## GitHub Pages

1. Upload these files to a GitHub repository.
2. In GitHub, open **Settings > Pages**.
3. Set **Source** to deploy from the default branch and choose the repository root.
4. Save and wait for GitHub Pages to publish the site.

## Notes

Microsoft 365 is configurable by organization, license, administrator settings, app availability, and security policy. Review this guide against your internal Teams, OneDrive, SharePoint, retention, sharing, guest access, and sensitivity-label rules before publishing it as official procedure.
